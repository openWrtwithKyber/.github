# 🧑‍🎓TEAM 십만전자
- 해당 프로젝트는 대구가톨릭대학교 2025년도 2학기 ~ 2026년도 1학기 캡스톤 디자인 프로젝트 입니다.
### 📌주제
- IoT 기기중 무선네트워크 공유기의 양자컴퓨터로 인한 공격 위협에 대비하기 위한 프로젝트입니다.
- 최신 프로토콜인 WPA3는 PQC를 위한 표준이 아직 존재하지 않아 해당 내용에 대한 제안 및 구현이 목표입니다.
### 🏃수행과정
- 오픈소스 공유기 펌웨어인 OpenWrt를 수정해 기존 WPA3-SAE와 하이브리드 양자내성암호를 구현합니다.
- 완성된 펌웨어를 컴파일 후 적용시켜 보안성 및 가용성을 검토합니다.
### 🌏사용환경
- 공유기 : ipTIME AX3000sm (AX3000se)
  - RAM : DDR3 256MB
  - NAND Flash : 128MB
- 펌웨어 버전 : 24.10.03 (w1.fi : `5ace39b0a4cdbe18ddbc4e18f80ee3876233c20b`)
- 빌드 환경 : WSL ubuntu 22.04
- 사용 암호 : ML-KEM (FIPS 203)

- 새로운 네트워크 표준을 제안합니다. 어떤 관리프레임을 사용하여 PQC를 운용할지 설계합니다.
